Hi There!

This webapp has been created to collect some data for upcoming Tasks Track. There are two types of labels we encourage you to provide.

First, task labels for 2 queries. This is to group queries into two classes : same task or different task. For example, 'london hotels' and 'barcelona hotels' are queries for same task 'hotels'. Whereas, 'sudoku' and 'pret menu' are queries that belong to different tasks.

Second, for a given entity, a starting query, we ask you to list follow up tasks to that query that come to your mind. It could from a very specific to a very generic task. For example, for entity 'madagascar', I could submit following tasks :'travel to madagascar', 'weather in madagascar', 'population of madagascar'.

There is no correct answer here, the app will show you statistics of other users too! So, engage with the app and enjoy!!
